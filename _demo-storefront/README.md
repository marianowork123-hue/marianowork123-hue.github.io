# Demo de la tienda (un solo archivo)

`azul-marino-storefront.jsx` es una **maqueta interactiva** de toda la tienda en un único
componente de React (catálogo, carrito, checkout simulado). Sirve para enseñar el diseño
sin instalar nada.

NO es la tienda real: no procesa pagos ni guarda datos. La tienda funcional es el
proyecto Next.js de la carpeta raíz. Esta maqueta queda solo como referencia visual.
